## 新建机器人
1、与BotFather启动对话
- 打开Telegram应用程序或网页版本
- 在搜索栏中搜索@BotFather或者跟随链接 https://t.me/BotFather
- 通过点击Start按钮与BotFather的聊天

2、创建一个新的机器人
- 向BotFather发送 /newBot 命令
- BotFather要求你为你的机器人选择一个名字。这是一个显示名称，可以包含空格。
- 为机器人选择一个用户名。必须以 bot 结尾并且是唯一的。

3、设置机器人小程序
- 向BotFather发送 /myBots 命令
